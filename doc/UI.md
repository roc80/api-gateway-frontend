## UI设计

### 背景

本项目是一个 **API 在线调用平台**，简单模仿 Apifox、Postman、Easy YAPI 等产品，提供接口定义、版本管理、在线调试等功能，**仅用于学习。**

### 设计方案：主从嵌套结构

参考 **Apifox** 的设计，采用 **左侧接口树 + 右侧详情面板** 的布局：

```
┌─────────────────────────────────────────────────────────────────┐
│  API 在线调用平台                                    [搜索框]     │
├──────────────┬──────────────────────────────────────────────────┤
│              │                                                   │
│  接口列表    │  右侧详情区（Tab 切换）                           │
│              │                                                   │
│  📁 用户类    │  ┌─────────────────────────────────────────┐   │
│    ├ 获取用户 │  │ 📋 接口详情                              │   │
│    └ 创建用户 │  │   名称: 获取用户信息                      │   │
│              │  │   路径: GET /api/user/{id}                │   │
│  📁 订单类    │  │   描述: ...                              │   │
│    ├ ...     │  └─────────────────────────────────────────┘   │
│              │                                                   │
│  [ + 新建 ]  │  ┌─────────────────────────────────────────┐   │
│              │  │ 🔧 在线调试                              │   │
│              │  │   [参数输入区]                           │   │
│              │  │   [发送按钮]                             │   │
│              │  └─────────────────────────────────────────┘   │
│              │                                                   │
│              │  ┌─────────────────────────────────────────┐   │
│              │  │ 📝 版本管理                              │   │
│              │  │   v1.0 (当前)  v1.1  v2.0  [+ 新建]     │   │
│              │  └─────────────────────────────────────────┘   │
└──────────────┴──────────────────────────────────────────────────┘
```

### 具体交互流程

#### 1. 主页面布局

```
页面路由: /api/workspace

布局:
  - 左侧: 接口树（按 category 分组，可折叠）
  - 右侧: 动态内容区（默认显示接口列表或第一个接口的详情）
```

#### 2. 用户交互流程

```
1. 用户进入 /api/workspace
   ↓
2. 左侧显示接口树（可折叠分类）
   - 📁 用户类 (3)
   - 📁 订单类 (5)
   - 📁 支付类 (2)
   ↓
3. 点击某个接口 → 右侧显示该接口的详情 Tabs
   ↓
4. Tabs 内容：
   ┌─────────────────────────────────────────────┐
   │ 📋 概览  │ 🔧 在线调试  │ 📝 版本管理  │ 📊 调用日志 │
   └─────────────────────────────────────────────┘

   各 Tab 功能：
   - 📋 概览: 接口基本信息 + 当前版本详情
   - 🔧 在线调试: 参数表单 + 发送按钮 + 响应展示
   - 📝 版本管理: 版本列表 + 切换当前版本 + 新建版本
   - 📊 调用日志: 历史调用记录（时间、状态、耗时）
```

#### 3. 在线调试交互

```
┌─────────────────────────────────────────────┐
│  在线调试                                    │
├─────────────────────────────────────────────┤
│                                             │
│  请求方法:  [GET ▼]                         │
│  请求路径:  /api/user/{id}                  │
│                                             │
│  Path 参数:                                 │
│  ┌─────────┬─────────┬────────────┐       │
│  │  参数名  │  类型   │   示例值    │       │
│  ├─────────┼─────────┼────────────┤       │
│  │  id     │ string  │  123       │       │
│  └─────────┴─────────┴────────────┘       │
│                                             │
│  [ 发送请求 ]    [ 保存为用例 ]             │
│                                             │
│  响应结果:                                  │
│  ┌─────────────────────────────────────┐  │
│  │ HTTP 200 OK (125ms)                  │  │
│  │ {                                   │  │
│  │   "code": 0,                         │  │
│  │   "data": { ... }                    │  │
│  │ }                                    │  │
│  └─────────────────────────────────────┘  │
└─────────────────────────────────────────────┘
```

#### 4. 版本管理交互

```
┌─────────────────────────────────────────────┐
│  版本管理                                    │

│                                             │
│  当前版本: v1.2                              │
│                                             │
│  所有版本:                                  │
│  ┌─────────────────────────────────────┐  │
│  │ v1.2  [当前]  [查看] [切换]          │  │
│  ├─────────────────────────────────────┤  │
│  │ v1.1         [查看] [切换] [删除]    │  │
│  ├─────────────────────────────────────┤  │
│  │ v1.0         [查看] [切换] [删除]    │  │
│  └─────────────────────────────────────┘  │
│                                             │
│  [+ 新建版本]                               │
└─────────────────────────────────────────────┘
```

### 前端页面结构

```
src/pages/api/
├── workspace/              # 主工作区（新建）
│   └── index.tsx          # 主页面（左右分栏布局）
│
├── components/            # 公共组件（新建）
│   ├── ApiTree.tsx       # 左侧接口树组件
│   ├── ApiDetailTabs.tsx # 右侧详情 Tabs 容器
│   ├── ApiOverviewTab.tsx    # 概览 Tab
│   ├── ApiDebugTab.tsx       # 在线调试 Tab
│   ├── ApiVersionTab.tsx     # 版本管理 Tab
│   ├── ApiCallLogTab.tsx     # 调用日志 Tab
│   └── RequestForm.tsx       # 请求参数表单
│
├── api-list/              # 独立列表页（保留或废弃）
└── api-version-list/      # 独立版本页（保留或废弃）
```
